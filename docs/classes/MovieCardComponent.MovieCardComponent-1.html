toggleFavorite

  • toggleFavorite(movieID: string): void
    • Defined in movie-card/movie-card.component.ts:163

    this function checks if a favorite movie icon is activated or not by movieID; If a user clicks on a empty icon, a POST request via FetchApiDataService.addFavoriteMovie is activated. That changes the movie favorite icon from emtpy to filled, set the movieID to the favorite list and a confirmation message will be displayed.

    If a user clicks on a filled icon, a DELETE request via FetchApiDataService.removeFavoriteMovie is activated. That changes the movie favorite icon from filled in to empty, removes the movieID to the favorite list and a confirmation message will be displayed.

    Parameters

    • movieID: string

    Returns void
